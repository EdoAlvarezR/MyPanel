# MyPanel
Three-dimensional panel method for high-Reynolds aerodynamics.

# Dependencies
  * VTKtools: `https://github.com/byuflowlab/VTKtools.jl`

# Installation
```julia
Pkg.clone("git@github.com:EdoAlvarezR/MyPanel.jl.git")
```
